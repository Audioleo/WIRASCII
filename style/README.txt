Archivo: style/README.txt
Propósito: Documentación de reglas, convenciones y estilos para el sistema WIRASCII.

Esta carpeta contiene archivos que definen las reglas oficiales y recomendaciones para la creación y representación
de símbolos, conexiones y esquemas eléctricos en WIRASCII.

Archivos típicos:
- wiring-rules.txt : Reglas para representación de cableado (wire units).
- naming-conventions.txt : Convenciones de nombres para símbolos.
- formatting-guidelines.txt : Reglas de formato para archivos y símbolos.

Es recomendable consultar estos archivos antes de crear o modificar símbolos o esquemas para asegurar
consistencia y calidad en el repositorio.
